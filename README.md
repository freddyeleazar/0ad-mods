# Mods para 0 A.D.

Sitio web con mods gratuitos para el juego de estrategia [0 A.D.](https://play0ad.com/), creados por **freddyeleazar**.

## 🎮 ¿Qué es 0 A.D.?

0 A.D. es un juego de estrategia en tiempo real gratuito y de código abierto, ambientado en la antigüedad. Es similar a Age of Empires y está lleno de historia y civilizaciones fascinantes.

## 🛠️ Mods Disponibles

### Day/Night Cycle
Añade un reloj de ciclo día/noche al HUD mostrando la fase actual y el tiempo virtual.

### Personal Names
Añade nombres personales únicos a las unidades humanas según su género y civilización.

### Experience Skills
Las unidades mejoran sus habilidades con la práctica. Las unidades civiles recolectan más rápido, las militares hacen más daño, y todas ganan vida máxima al recibir daño.

## 📥 Cómo Descargar

1. Haz clic en "Ver detalles" del mod que quieres
2. Haz clic en el botón de descarga
3. Sigue las instrucciones de instalación

## 📋 Requisitos

- **Juego**: 0 A.D. versión 0.28.0 o superior
- **Plataforma**: Windows, Linux, Mac
- **Idioma**: Español
